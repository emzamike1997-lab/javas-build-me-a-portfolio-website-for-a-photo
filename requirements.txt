### Running the Application
To run the application, navigate to the `backend` directory and run the following command:
```bash
python app.py
```
This will start the Flask development server, and you can access the API endpoints by visiting `http://localhost:5000` in your web browser.

### API Endpoints
The following API endpoints are available:

* `GET /photographers`: Get all photographers
* `GET /photographers/<int:photographer_id>`: Get photographer by id
* `GET /photos`: Get all photos
* `GET /photos/<int:photo_id>`: Get photo by id
* `POST /photographers`: Create new photographer
* `POST /photos`: Create new photo

Note: This is a basic implementation, and you may want to add additional error handling, validation, and security measures depending on your specific use case.
